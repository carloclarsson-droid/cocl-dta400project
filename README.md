This repository hosts code, configuration, and specifications for this project.

VB-debian-ns3-specifications.txt
Hardware and software information for virtual box, debian, and ns-3 network simulator.

modified-ns3-code.cc
The modified version of "simple-routing-ping6.cc" used for this project.

physical-configuration-specification.txt
Configuration for physical nodes, along with hardware and software specifications. Ping command used is also listed here. 

physical-ping-output.txt
Ping output as retrieved from the command prompt of node0.

r-running-configuration.txt
Output from router listing its running configuration.

r-version.txt
Output from router listing its version and other software information.

retrieve-latency.py
Simple python program used to clean outputs by retrieving latency values.

simulated-ping-output.txt
Output from executing the program "modified-ns3-code.cc". The name in the output differs, but the code is the same.
